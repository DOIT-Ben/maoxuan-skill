4. 你打你的，我打我的
5. 打得赢就打，打不赢就走

### 第2天：根据问题类型查阅Skill

## 使用注意事项

### 1. 反对教条主义

本skill提供的是方法论框架，具体应用时必须结合实际情况，不能生搬硬套，不能本本主义。

**正确做法**：
- 具体问题具体分析
- 根据实际情况调整
- 在实践中检验和完善

### 2. 坚持实事求是

所有决策必须建立在充分调查研究的基础上，不能凭想当然办事，不能脱离实际。

**正确做法**：
- 先调查，后决策
- 用事实说话
- 让实践检验

### 3. 持续学习

毛选智慧博大精深，本skill只是入门指引，建议阅读原著，深入理解精神实质。

**推荐做法**：
- 定期重温经典篇目
- 结合实践反复体会
- 与他人交流心得

### 4. 实践检验

理论是否正确，最终要靠实践检验，要在实践中不断总结、完善、提高。

**正确态度**：
- 勇于实践
- 善于总结
- 不断改进
