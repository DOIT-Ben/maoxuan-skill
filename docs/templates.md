| 序号 | 矛盾描述 | 类型 | 主次判断 | 主要方面 | 解决思路 |

### 填表说明
- **矛盾描述**：用"[A] vs [B]"格式描述矛盾双方
- **类型**：对抗性（如利益根本冲突）/ 非对抗性（如方法分歧）
- **主次判断**：标注"主要"或"次要"
- **主要方面**：矛盾中占支配地位的一方
- **解决思路**：针对性的解决方法

### 示例

| 序号 | 矛盾描述 | 类型 | 主次判断 | 主要方面 | 解决思路 |

| 2 | 进度要求 vs 质量标准 | 非对抗性 | 次要 | 质量标准 | 坚守质量底线，协商进度 |
| 3 | 成本控制 vs 功能完整 | 非对抗性 | 次要 | 功能完整 | MVP优先，后续迭代 |

## 模板三：持久战战略规划表

### 使用说明
面对长期复杂任务时，使用此表进行阶段性规划。

### 模板内容

| 阶段 | 时间 | 目标 | 策略 | 关键任务 | 风险预判 |
|------|------|------|------|---------|---------|
| 战略防御 | | 生存下来 | 保存实力 | | |
| 战略相持 | | 积累力量 | 积极发展 | | |
| 战略反攻 | | 夺取胜利 | 决战决胜 | | |

### 阶段判定标准

**战略防御期特征**：
- 敌强我弱，力量悬殊
- 主要任务是保存实力，避免决战
- 行动原则：打得赢就打，打不赢就走

**战略相持期特征**：
- 力量对比趋于平衡
- 主要任务是积累力量，积极发展
- 行动原则：扩大根据地，发动群众

**战略反攻期特征**：
- 我强敌弱，优势确立
- 主要任务是决战决胜，夺取胜利
- 行动原则：集中优势兵力，歼灭敌人

## 模板四：统一战线分析表

### 使用说明
在需要团结多方力量时，使用此表分析各类势力。

### 模板内容

| 势力 | 特征 | 态度 | 策略 | 具体行动 |
|------|------|------|------|---------|
| 进步势力 | | | 大力发展 | |
| 中间势力 | | | 积极争取 | |
| 顽固势力 | | | 孤立分化 | |

### 三种势力定义

**进步势力**：
- 坚定支持我方
- 是基本依靠力量
- 策略：大力发展，扩大队伍

**中间势力**：
- 摇摆不定，可左可右
- 需要积极争取
- 策略：加强沟通，争取支持

**顽固势力**：
- 反对我方，但不是主要敌人
- 需要分化孤立
- 策略：利用矛盾，各个击破

### 又联合又斗争原则
- 联合是目的，斗争是手段
- 以斗争求团结则团结存
- 有理、有利、有节
