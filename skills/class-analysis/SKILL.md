# 阶级分析专家

## 核心原理

> "谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。" —— 《中国社会各阶级的分析》

**本skill服务于核心原理一：分清敌我友——界定问题**

## 什么是阶级分析

遇到问题时，首先问：**这是什么性质的问题？属于哪一类？**

然后区分：
- **需要解决的问题**：必须处理的
- **不需要解决的问题**：可以放一放的
- **问题的类别**：技术问题、组织问题、资源问题...

## 分析步骤

### 第一步：划分问题类别

将问题分类：

| 类别 | 特征 | 策略 |
|------|------|------|
| 技术问题 | 技术层面的困难 | 技术攻关 |
| 组织问题 | 人和流程的问题 | 组织调整 |
| 资源问题 | 资源不足的问题 | 资源争取 |
| 认知问题 | 认识和理解的问题 | 教育沟通 |

### 第二步：判断紧迫程度

| 紧迫程度 | 特征 | 策略 |
|---------|------|------|
| 紧急重要 | 影响核心目标 | 立即处理 |
| 重要不紧急 | 影响长期发展 | 计划处理 |
| 紧急不重要 | 影响当前效率 | 委托处理 |
| 不紧急不重要 | 无实际影响 | 暂不处理 |

### 第三步：确定处理优先级

明确：
1. 什么必须现在解决
2. 什么可以之后解决
3. 什么不需要解决

## 输出格式

```
问题类别：_______
紧迫程度：[紧急重要/重要不紧急/紧急不重要/不紧急不重要]

必须现在解决：_______
可以之后解决：_______
不需要解决：_______
```

## 与其他skill的关系

- **矛盾分析**：在分类基础上找主要矛盾
- **调查研究**：深入了解各类问题的具体情况
- **根据地建设**：找到突破口后分步解决
