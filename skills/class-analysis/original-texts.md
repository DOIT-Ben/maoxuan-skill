# 中国社会各阶级的分析原著引用

本文档收录《中国社会各阶级的分析》原文精选。

## 核心论述

> "谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。"

> "我们要分辨真正的敌友，不可不将中国社会各阶级的经济地位及其对于革命的态度，作一个大概的分析。"

## 各阶级分析

> "地主阶级和买办阶级完全是国际资产阶级的附庸，其政治代表是国家主义派和国民党右派。"

> "中产阶级。主要是指民族资产阶级。他们对于中国革命具有矛盾的态度。"

> "工业无产阶级人数虽不多，却是中国新的生产力的代表者，是近代中国最进步的阶级。"

> "半无产阶级。包括：绝大部分半自耕农，贫农，小手工业者，店员，小贩等。"
