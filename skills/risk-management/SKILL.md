# 风险预判与危机管理

## 核心原理

> "我们则是马克思主义者，马克思主义叫我们看问题不要从抽象的定义出发，而要从客观存在的事实出发，从分析这些事实中求出方针、政策和办法来。" —— 《反对党八股》

**本skill服务于核心原理一：分清敌我友——界定问题**

## 什么是底线思维

遇到问题时，首先问：
1. **最坏的情况是什么** — 预设风险
2. **如何避免最坏情况** — 预防措施
3. **如果发生怎么办** — 应急预案

## 分析步骤

### 第一步：识别风险

可能的风险有哪些？
- 最坏的情况是什么
- 发生的概率如何

### 第二步：评估影响

风险的影响有多大？
- 对核心目标的影响
- 对其他方面的影响

### 第三步：制定预案

如何应对风险？
- 预防措施
- 应急预案

## 输出格式

```
风险识别：
1. 风险1：_______
   - 最坏情况：_______
   - 发生概率：_______
   - 影响程度：_______

2. 风险2：_______
   - ...

应急预案：
- 预防措施：_______
- 响应措施：_______
- 恢复措施：_______
```

## 与其他skill的关系

- **调查研究**：了解可能的风险
- **矛盾分析**：分析风险的主要矛盾
- **根据地建设**：做好风险预案
