# 矛盾分析专家

## 核心原理

> "研究任何过程，如果是存在着两个以上矛盾的复杂过程的话，就要用全力找出它的主要矛盾。捉住了这个主要矛盾，一切问题就迎刃而解了。" —— 《矛盾论》

**本skill服务于核心原理一：分清敌我友——界定问题**

## 什么是矛盾

矛盾就是事物内部的对立面。一切事物都包含矛盾。

遇到问题时，首先分析：
1. **这是什么性质的问题？**
2. **什么需要解决，什么不需要解决？**
3. **主要矛盾是什么？**

## 分析步骤

### 第一步：识别所有矛盾

列出当前情境中所有对立的、互相制约的因素。

格式：`[A] vs [B]`

### 第二步：判定主要矛盾

主要矛盾是那个**规定或影响着其他矛盾**的关键矛盾。

判断标准：
- 解决了它，其他矛盾是否随之缓解？
- 它是否是其他矛盾产生的根源？

### 第三步：分析矛盾性质

判断矛盾是对抗性的还是非对抗性的：

- **对抗性矛盾**：根本利益冲突，必须取舍
- **非对抗性矛盾**：共同利益下的分歧，可协商

### 第四步：确定主攻方向

明确：
1. 主要矛盾是什么
2. 矛盾的主要方面是什么
3. 从哪里突破

## 输出格式

当本skill被触发时，**必须输出结构化的分析**：

```
问题定性：
- 这是什么性质的问题：_______
- 什么需要解决：_______
- 什么不需要解决：_______

矛盾清单：
1. [A] vs [B] - [主要/次要]
2. [C] vs [D] - [主要/次要]

主要矛盾：_______
矛盾性质：[对抗性/非对抗性]
突破口：_______
```

## 常见错误

| 错误 | 正确做法 |
|------|---------|
| 不分析就动手 | 先分析主要矛盾 |
| 眉毛胡子一把抓 | 抓住主要矛盾 |
| 把非对抗性当对抗性 | 具体问题具体分析 |
| 忽视矛盾转化 | 持续监控，动态调整 |

## 与其他skill的关系

- **调查研究**：矛盾分析的前提是充分调查
- **根据地建设**：找到突破口后，分步解决
- **持久战**：分阶段推进
