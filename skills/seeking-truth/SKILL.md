# 实事求是决策师

## 核心原理

> "'实事'就是客观存在着的一切事物，'是'就是客观事物的内部联系，即规律性，'求'就是我们去研究。" —— 《改造我们的学习》

**本skill服务于核心原理一：分清敌我友——界定问题**

## 什么是实事求是

1. **从实际出发** — 不是从愿望出发
2. **让事实规定判断** — 不是让判断规定事实
3. **让现实修正理论** — 不是让理论修正现实

## 分析步骤

### 第一步：承认未知

先承认"我不知道什么"：
- 我目前掌握的事实是_______
- 我还不知道的是_______
- 需要进一步调查的是_______

### 第二步：调查求实

深入调查，获取第一手资料

### 第三步：得出判断

基于事实得出判断：
- 实际情况是_______
- 本质规律是_______
- 我的判断是_______

## 判断准则

| 正确做法 | 错误做法 |
|---------|---------|
| 从实际出发 | 从愿望出发 |
| 让事实规定判断 | 让预想规定事实 |
| 实践检验真理 | 理论脱离实际 |

## 输出格式

```
已知与未知：
- 已掌握的事实：_______
- 未知领域：_______
- 需要进一步调查的内容：_______

调查结论：
- 实际情况：_______
- 本质规律：_______
- 基于事实的判断：_______

判断记录：
| 判断内容 | 事实依据 | 结论 |
|---------|---------|------|
| _______ | _______ | _______ |
```

## 与其他skill的关系

- **调查研究**：实事求是的基础
- **矛盾分析**：基于事实分析矛盾
- **根据地建设**：从实际情况出发制定方案
