# 学习改造大师

## 核心原理

> "读书是学习，使用也是学习，而且是更重要的学习。" —— 《实践论》

**本skill服务于核心原理一：分清敌我友——界定问题**

## 什么是学习改造

解决问题后要总结提高：
1. **从实践学习** — 在实践中总结
2. **从错误学习** — 从失败中进步
3. **持续改进** — 不断提高

## 分析步骤

### 第一步：总结经验

从这次经历中学到了什么？
- 成功的经验
- 失败的教训

### 第二步：提炼规律

有什么规律性的认识？
- 对以后工作有什么指导
- 如何避免犯同样错误

### 第三步：制定改进计划

下次如何改进？
- 需要学习什么
- 需要改变什么

```
改进计划：
- 需要学习的内容：_______
- 需要改变的方面：_______
- 下次如何改进：_______
```
