# 工作流组合

本skill围绕两个核心原理展开：

- **原理一**：分清敌我友——界定问题
- **原理二**：星星之火，可以燎原——分步解决

---

## 工作流一：问题界定

**核心原理**：原理一（分清敌我友）

**适用场景**：遇到新问题，需要分清性质和主次时

**流程**：
```
调查研究 → 矛盾分析 → 问题定性 → 确定主攻方向
```

**每一步的目的**：
1. **调查研究**：深入了解问题的真实情况
2. **矛盾分析**：找出主要矛盾
3. **问题定性**：这是什么问题？什么需要解决？
4. **确定主攻方向**：明确突破口

---

## 工作流二：分步攻坚

**核心原理**：原理二（星星之火，可以燎原）

**适用场景**：问题庞大，需要分步解决时

**流程**：
```
找出根源 → 拆分问题 → 确定突破口 → 分步解决 → 积小胜为大胜
```

**每一步的目的**：
1. **找出根源**：找到问题的根本原因
2. **拆分问题**：将大问题拆分为小问题
3. **确定突破口**：从最容易解决的地方入手
4. **分步解决**：一步步解决
5. **积小胜为大胜**：从小胜积累到大胜

---

## 工作流三：迭代优化

**核心原理**：原理一 + 原理二

**适用场景**：持续改进已有方案时

**流程**：
```
调查现状 → 分析矛盾 → 制定方案 → 执行检验 → 复盘改进
```

**每一步的目的**：
1. **调查现状**：了解当前方案的效果
2. **分析矛盾**：找出主要问题
3. **制定方案**：制定改进方案
4. **执行检验**：实施方案并验证
5. **复盘改进**：总结经验，持续改进

---

## 工作流四：统一战线

**核心原理**：原理一（分清敌我友）

**适用场景**：需要整合资源、扩大力量时

**流程**：
```
分析力量 → 团结友军 → 孤立敌人 → 扩大战果
```

**每一步的目的**：
1. **分析力量**：识别各方势力立场
2. **团结友军**：巩固同盟
3. **孤立敌人**：分化瓦解
4. **扩大战果**：扩大统一战线

---

## 工作流选择指南

| 你的情况 | 推荐工作流 |
|---------|---------|
| 遇到新问题，需要分清性质 | 工作流一：问题界定 |
| 问题庞大，需要分步解决 | 工作流二：分步攻坚 |
| 持续改进已有方案 | 工作流三：迭代优化 |
| 需要整合资源扩大力量 | 工作流四：统一战线 |

## 数据传递规则

工作流中每个步骤的**输出必须显式传递给下一步骤**：

1. 每个步骤结束时，写出格式化的"传递给下一步的信息"
2. 下一步开始时，确认收到并引用上一步的输出
3. 不得跳步骤执行

## 工作流中断处理

如果在工作流执行过程中遇到阻塞：
- **信息不足** → 插入一次调查研究，补充信息后继续
- **假说被证伪** → 返回矛盾分析，重新分析
- **发现新的更重要的矛盾** → 重新界定问题
