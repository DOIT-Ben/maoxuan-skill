# 调查研究专员

## 核心原理

> "没有调查，没有发言权。" —— 《反对本本主义》

**本skill服务于核心原理一：分清敌我友——界定问题**

在判断问题性质之前，首先需要深入了解真实情况。

## 调查研究的目的是什么

1. **了解真实情况** — 不是表面现象
2. **为判断提供依据** — 让结论有事实支撑
3. **分清问题性质** — 为矛盾分析做准备

## 分析步骤

### 第一步：明确调查目的

你要弄清楚什么？
格式：我要弄清楚的是_______


### 第二步：列出调查清单

需要调查哪些方面？
- 现状是什么
- 历史演变
- 各方意见
- 约束条件

### 第三步：深入调查

采用合适的调查方法：
- 开调查会
- 个别访谈
- 实地考察
- 文献研究

### 第四步：整理结论

```
调查结论：
- 现状是：_______
- 关键约束：_______
- 我之前不知道现在知道的是：_______
- 基于以上，问题定性为：_______
```

## 调查要领

1. **眼睛向下** — 不要只看表面
2. **全面系统** — 防止片面性
3. **深入细致** — 抓住本质

## 与其他skill的关系

- **矛盾分析**：调查结论是矛盾分析的基础
- **阶级分析**：调查为问题定性提供依据
- **根据地建设**：调查后找到突破口
