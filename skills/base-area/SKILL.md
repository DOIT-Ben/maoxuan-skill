# 根据地建设者

## 核心原理

> "星星之火，可以燎原。" —— 《星星之火，可以燎原》

**本skill服务于核心原理二：星星之火，可以燎原——分步解决**

## 什么是根据地

**问题庞大时，不要想着一口吃成胖子，而是要找到切入点，像建立根据地一样，一步步扩展。**

### 根据地的三个特征：
1. **切入点**：从最容易突破的地方入手
2. **稳固性**：建立稳定的基础
3. **扩展性**：从点到线到面，波浪式推进

## 分析步骤

### 第一步：选择切入点

找到最容易突破的地方作为"根据地"：
- 问题中最容易解决的部分
- 能产生"小胜"的部分
- 能建立信心和势能的部分

### 第二步：建立稳固基础

在切入点建立稳固的基础：
- 彻底解决这个问题
- 验证方法的有效性
- 积累经验和资源

### 第三步：波浪式推进

从点到线到面，逐步扩展：
```
第一步（点）：解决切入点问题
         ↓
第二步（线）：扩展到相关问题
         ↓
第三步（面）：最终燎原，解决整个问题
```

## 关键原则

1. **先易后难**：先从容易的入手
2. **彻底解决**：不求快但求彻底
3. **积小胜为大胜**：小胜积累成大胜
4. **波浪式推进**：点→线→面

## 与其他skill的关系

- **矛盾分析**：找到主要矛盾后，用此skill分步解决
- **持久战**：分阶段推进的战略框架
- **灵活机动**：在运动中寻找机会
